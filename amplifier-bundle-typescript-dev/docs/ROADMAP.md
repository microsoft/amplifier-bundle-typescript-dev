# TypeScript-dev Bundle Roadmap

## Current (v0.1.0)

- Prettier formatting checks
- ESLint linting (v8 and v9 JSON output)
- TSC type checking (project-wide)
- Stub/placeholder detection (TODO, FIXME, @ts-ignore, etc.)
- Automatic checking hook on file write/edit
- TypeScript + JavaScript LSP via typescript-language-server
- Two agents: code-intel (LSP) + typescript-dev (quality)

## Future

- [ ] Biome support (alternative to prettier + eslint)
- [ ] Deno support (alternative TypeScript runtime)
- [ ] Vue/Svelte SFC support
- [ ] Custom ESLint rule recommendations
- [ ] Performance profiling integration
- [ ] Test runner integration (jest, vitest)
